# company_control_program

### 프로젝트 설명

2학년 2학기 자바 프로그래밍 시간에 진행했던 프로젝트입니다.
프로젝트를 중점적으로 하는 회사에서는 어떻게 프로세스가 이루어질까 고민하며 만들었습니다.

### 사용한 기술 스택

Swing, java, socket, mysql
